What Did we Learn In IST250?

HTML

(HyperText Markup Language) is the standard language used to create and structure content on the web. It uses a system of tags and elements to define how text, images, links, and other media are displayed in a browser.

CSS

(Cascading Style Sheets) is a styling language used to control the look and feel of web pages. It works alongside HTML to define how elements—such as text, images, and layouts—are displayed, including colors, fonts, spacing, and positioning. By separating content from design, CSS makes it easier to maintain, update, and create consistent styles across multiple pages, enabling responsive and visually appealing websites.

JavaScript

A versatile, high-level programming language that brings interactivity and dynamic behavior to websites and web applications. It runs in all major browsers, enabling developers to create features like animations, form validations, real-time updates, and interactive user interfaces. Beyond the browser, JavaScript is also used on the server side (via environments like Node.js) to build scalable, full-stack applications. Its flexibility, vast ecosystem of libraries, and active community make it one of the most essential tools in modern web development.

Website Hosting

A digital service that enables others to see and use your website online. This service stores your website’s text, photos, videos, files and other data on a physical server somewhere so that others can access it via the internet at any time.
